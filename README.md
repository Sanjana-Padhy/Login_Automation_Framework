Login Automation Framework

Overview

Login Automation Framework is a Selenium WebDriver and TestNG-based automation testing project developed using Java and Maven. The project automates login validation scenarios and demonstrates the implementation of automated functional testing for web applications.

This project covers positive and negative login test scenarios using Selenium WebDriver, TestNG assertions, and Maven dependency management.


Technologies Used:
* Java
* Selenium WebDriver
* TestNG
* Maven
* WebDriverManager
* Eclipse IDE
* Git & GitHub


Features:
* Automated Login Validation
* Positive Login Test Case
* Invalid Password Test Case
* Invalid Username Test Case
* Assertion-Based Verification
* Automated Browser Launch and Closure
* Maven Dependency Management

Test Scenarios Implemented

1. Valid Login Test
* Open Login Page
* Enter valid username
* Enter valid password
* Click Login
* Verify successful login


2. Invalid Password Test
* Enter valid username
* Enter incorrect password
* Click Login
* Verify error message


3. Invalid Username Test
* Enter invalid username
* Enter valid password
* Click Login
* Verify error message
